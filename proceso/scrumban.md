# Scrumban 

Modelo de proceso de desarrollo de software híbrido que surge de la combinación de prácticas entre [Scrum](scrum.md) y [Kanban](kanban.md).

El trabajo realizado implica modificaciones sobre [ítems de configuración](../gcs/item-de-configuracion.md). Estos cambios son gestionados mediante el proceso de [Gestión de Configuración del Software](../gcs/gcs.md).

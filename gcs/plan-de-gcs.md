# Plan de Gestión de Configuración del Software (Plan de GCS)

Es el documento resultante de la fase de [planeamiento](planeamiento-de-la-gcs.md) cuyo objetivo principal es establecer las actividades, los responsables y los recursos necesarios para llevar a cabo la [GCS](gcs.md) en un proyecto específico. 

Estructura el contexto organizacional, el alcance, los tipos de elementos que serán controlados (futuros [ítems de configuración](item-de-configuracion.md)), el cronograma, y las herramientas a utilizar.
